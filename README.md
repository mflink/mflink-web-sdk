## install plugin

### Mac
```
unzip downloads/osx/npFlintPlugin.plugin.zip
cp -r downloads/osx/npFlintPlugin.plugin ~/Library/Internet\ Plug-Ins/
```

### Linux
```
mkdir -p ~/.mozilla/plugins/
cp downloads/linux/npFlintPlugin.so ~/.mozilla/plugins/
```
